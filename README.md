# Websocket Client


## Introduction
React focused example of a websocket client hosted on port 10100.


## How To
### Install dependencies
Using `git bash` or equivalent execute:
```
$ yarn install
```

### Build the Web App
Using `git bash` or equivalent execute:
```
$ yarn build
```

### Run the Web App
Using `git bash` or equivalent execute:
```
$ yarn start
```
